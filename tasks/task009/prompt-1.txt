在当前 Vue 3 + Vite 项目中实现邀请返利页面，覆盖邀请码分享、邀请记录、奖励规则、到账状态、异常冻结说明和提现入口。
1. 页面需要展示用户的邀请码、分享入口和可复制的邀请链接，并在复制或分享后给出反馈。
2. 邀请记录需要展示被邀请人、邀请时间、转化状态、奖励金额和到账状态。
3. 奖励规则需要清晰说明返利条件、结算周期、冻结原因和提现限制。
4. 异常冻结记录需要展示冻结说明和处理建议，不能只显示模糊状态。
5. 页面需要展示可提现金额、待到账金额和提现入口，提现不可用时需要说明原因。
6. 页面在桌面端和 375px 移动端宽度下都要保持可用，规则说明、记录列表和金额卡片不能溢出。
7. 代码实现放在 src/pages/invite-rebate/ 下，测试代码放在 src/pages/__tests__/invite-rebate/ 下。
8. 需要为邀请码格式、奖励计算、到账状态、冻结原因和提现可用性相关的纯 JS 逻辑补充单元测试，并确保 pnpm run test 通过。
