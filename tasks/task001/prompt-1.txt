在当前 Vue 3 + Vite 项目中实现用户认证系统页面，覆盖登录、注册、找回密码、修改密码的完整流程，并使用本地状态模拟 Token 会话。
1. 页面需要支持登录态和未登录态切换，未登录时可访问登录、注册、找回密码，登录后可查看用户信息并进入修改密码流程。
2. 登录、注册、找回密码、修改密码都需要有清晰的表单校验、错误提示、提交反馈和流程跳转。
3. Token 会话需要支持持久化、退出登录、修改密码后会话失效，以及刷新页面后状态恢复。
4. 页面需要体现路由守卫或等价的访问限制效果，未登录用户不能直接操作需要登录态的功能。
5. 页面在桌面端和 375px 移动端宽度下都要保持可用，表单和导航不能挤压或溢出。
6. 代码实现放在 src/pages/auth/ 下，测试代码放在 src/pages/__tests__/auth/ 下。
7. 需要为表单校验、会话状态和访问限制相关的纯 JS 逻辑补充单元测试，并确保 pnpm run test 通过。
