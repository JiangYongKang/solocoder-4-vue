## ✨ 功能亮点

### 1. 步骤进度展示
- 进度条 + 步骤指示器双重展示
- 已完成步骤显示 ✓ 勾号，当前步骤高亮
- 百分比进度动态计算

### 2. 六个完整步骤
| 步骤 | 内容 |
|------|------|
| 👋 欢迎页 | 功能介绍卡片、动画效果 |
| ⚙️ 偏好设置 | 主题选择（浅色/深色/跟随系统）、语言（中英文）、通知方式（4种） |
| 📊 示例数据 | 开关切换、特性说明、启用状态提示 |
| ✅ 确认配置 | 摘要展示、每项可修改跳转、完成奖励预览 |
| ⏭️ 已跳过 | 后续建议操作、小贴士说明 |
| 🎉 完成页 | 快速统计、奖励卡片、推荐任务列表 |

### 3. 交互特性
- **跳过确认弹窗**：列出跳过后果 + 后续可配置说明
- **摘要修改**：确认页可直接点击"修改"跳回对应步骤
- **任务流转**：推荐任务可点击切换 `待办 → 进行中 → 已完成`
- **步骤回退**：所有中间步骤都可以返回上一步
- **重新查看**：完成或跳过状态都可重启引导流程

### 4. 响应式设计
- **桌面端**（>900px）：三列特性卡片、网格布局选项
- **平板端**（640-900px）：单列表格、双列通知选项
- **移动端**（375-640px）：隐藏步骤文字、单列全部选项、任务卡片纵向布局
- **375px**：完整适配、padding 缩减、字号调整、底部按钮不遮挡
