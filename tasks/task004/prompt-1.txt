在当前 Vue 3 + Vite 项目中实现新用户引导流程页面，帮助首次进入系统的用户完成基础偏好配置并理解后续任务。
1. 页面需要展示步骤进度，并支持在引导步骤之间前进、返回和完成。
2. 引导内容需要包含偏好选择、示例数据开关、跳过确认、完成奖励和后续任务推荐。
3. 用户选择偏好和示例数据开关后，页面需要保存当前选择并在确认页展示摘要。
4. 用户点击跳过时需要有确认提示，确认后进入跳过状态并说明后续可以继续配置。
5. 完成引导后需要展示奖励反馈和推荐任务入口，形成完整闭环。
6. 页面在桌面端和 375px 移动端宽度下都要保持可用，步骤条、选项卡片和底部按钮不能遮挡。
7. 代码实现放在 src/pages/onboarding/ 下，测试代码放在 src/pages/__tests__/onboarding/ 下。
8. 需要为步骤流转、偏好摘要、跳过确认和完成奖励相关的纯 JS 逻辑补充单元测试，并确保 pnpm run test 通过。
