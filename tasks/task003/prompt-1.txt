在当前 Vue 3 + Vite 项目中实现多租户空间切换页，帮助用户查看并切换自己所属的组织空间。
1. 页面需要展示组织空间列表，包含空间名称、成员身份、是否默认空间、是否过期和最近访问时间。
2. 用户可以切换当前空间，也可以将某个可用空间设置为默认空间，页面需要给出明确的状态反馈。
3. 过期空间需要有醒目的提醒，并限制或弱化不可用操作，避免用户误以为可以正常进入。
4. 页面需要提供邀请入口，并展示最近访问记录，方便用户快速回到常用空间。
5. 空列表、加载中、切换成功、切换失败等状态都需要有合理展示。
6. 页面在桌面端和 375px 移动端宽度下都要保持可用，空间卡片和操作区不能溢出。
7. 代码实现放在 src/pages/tenant-switch/ 下，测试代码放在 src/pages/__tests__/tenant-switch/ 下。
8. 需要为空间排序、默认空间设置、过期空间判断和最近访问记录相关的纯 JS 逻辑补充单元测试，并确保 pnpm run test 通过。
