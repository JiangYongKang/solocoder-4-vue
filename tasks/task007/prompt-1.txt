在当前 Vue 3 + Vite 项目中实现账号注销申请页，覆盖注销影响说明、身份二次验证、冷静期倒计时、撤销申请和进度查询。
1. 页面需要清晰展示账号注销会影响的数据、权益和不可逆风险，并要求用户确认已知晓。
2. 提交注销申请前需要完成身份二次验证，验证失败时给出明确错误提示。
3. 注销申请提交后进入冷静期状态，页面需要展示倒计时、当前进度和数据清除范围。
4. 冷静期内需要支持撤销申请，撤销后状态恢复并展示撤销成功反馈。
5. 页面需要支持查看注销进度，区分未申请、冷静期中、已撤销、已完成等状态。
6. 页面在桌面端和 375px 移动端宽度下都要保持可用，风险说明和进度信息不能挤压难读。
7. 代码实现放在 src/pages/account-cancellation/ 下，测试代码放在 src/pages/__tests__/account-cancellation/ 下。
8. 需要为二次验证、冷静期计算、撤销申请、数据清除范围和状态展示相关的纯 JS 逻辑补充单元测试，并确保 pnpm run test 通过。
