## ✅ 功能覆盖

| 需求 | 实现 |
|------|------|
| **1. 注销影响说明** | 5 大类（个人资料/订单/资产/权限/社交），3 种筛选 Tab（全部/不可逆/高风险），5 项不可逆风险卡片，必须勾选确认框才能进入下一步 |
| **2. 身份二次验证** | 3 种方式（密码/短信验证码/邮箱验证码），格式校验、错误次数达 5 次锁定、60s 验证码重发倒计时、过期判断，明确错误提示 |
| **3. 冷静期倒计时** | 天/时/分/秒实时倒计时（1s 刷新），进度百分比条、4 个阶段里程碑（提交→中期提醒→即将完成→注销完成），3 种清除范围分类展示 |
| **4. 撤销申请** | 5 种撤销原因选择，二次确认，撤销后展示时间线与撤销成功反馈，支持重新申请 |
| **5. 进度查询** | 4 种状态（未申请/冷静期中/已撤销/已完成）分别展示不同内容，时间自动流转到完成 |
| **6. 响应式适配** | 桌面端 max-width 960px，≤768px 响应式调整，≤375px 专门优化（字号/间距/卡片堆叠/按钮全宽） |
| **7. 代码路径** | 页面与测试均已放在要求目录 |
| **8. 单元测试** | 5 个纯逻辑测试文件，覆盖常量、数据影响、冷静期计算、身份验证、状态流转，共新增约 200 个用例，`pnpm run test` 全部 500 个用例通过 |

## 🔧 测试用例设计亮点

- **二次验证**：空值/短密码/非数字验证码/错误密码/达到最大尝试次数锁定等边界场景
- **冷静期计算**：开始/结束/中间时间点、非法日期、自定义天数、进度百分比边界（0%/50%/100%）
- **撤销申请**：各状态下撤销权限（仅冷静期可撤销）、重复提交保护
- **数据清除范围**：三种范围各分类的正确性、空列表、不可逆/可逆数量统计
- **状态展示**：状态机合法/非法流转、各状态描述/图标/颜色的一致性
